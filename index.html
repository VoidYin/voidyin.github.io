四、逻辑

形式逻辑是一门科学。研究对象是思维。
思维是人脑对客观的反映，而且必须以语言为载体。
思维形式：概念，判断，推理
思维形式结构：提取出的共同且不变的东西是规则和规律，包括常项和变项

语词 对应 概念
概念：反映某个事物
要求：具体明确
内涵：事物的本质特征
外延：一个个事物
反变关系：内涵越少，外延越多；内涵越多，外延越少。

逻辑方法：
限制：增加内涵，减少外延
概括：减少内涵，增加外延
定义：明确内涵。包括被定义项，定义项，联项。
划分：明确外延。包括母项，子项，根据

命题 对应 判断
判断：对概念的肯定或否定

命题：表达判断的句子
简单命题
复合命题
模态命题

推理：
由一个或多个判断推出另一个判断；
由一个或多个命题推出另一个新命题
包括前提，结论，推理形式。
要求：前提为真；推理形式为真
分为演绎，归纳，类比

（一）演绎推理
演绎分为：
简单命题：
1 性质命题：断定事物具有或不具有某种性质的命题。
包括主项，谓项，联项，量项
分为全称肯定，全称否定，特称肯定，特称否定
（1）直接推理：由一个推另一个
换质位
换质
所有A是B -> 所有A不是非B
所有A不是B -> 所有A是非B
有的A是B -> 有的A不是非B
有的A不是B -> 有的A是非B

换位：不周延的项，换位后也不能周延
所有A是B -> 有的B是A
所有A不是B -> 所有B不是A
有的A是B -> 有的B是A
有的A不是B 无法换位，因为换位后A从不周延变成周延

逻辑方阵

矛盾关系：必有一真一假。如果一个为真，另一个必为假；一个为假，另一个必为真。
反对关系：至少一假。如果一个为真，另一个必为假。
下反对关系：至少一真。如果一个为假，另一个必为真。
差等关系：上真下真，下假上假。

并非所有A是B = 有的A不是B
并非所有A不是B = 有的A是B
并非有的A是B = 所有A不是B
并非有的A不是B = 所有A是B
（2）三段论推理：由两个推另一个
小项：结论主项
大项：结论谓项
中项：结论没出现的项
要求：中项至少周延一次；前提中不周延，结论中也不周延；前提与结论否定数一样；前提之一是特称，结论必是特称。
2 关系命题：断定两个事物之间的关系
包括关系项，关系者项，量项
关系分为：对称关系，传递关系
关系推理分为：
纯关系推理，分为直接关系推理，一个推另一个；间接关系推理，两个推另一个。
混合关系推理，关系+性质

复合命题
1 联言命题：断定若干事物同时为真
对应语言：且
真傎表：当所有肢命题为真时，复合命题为真
推理：
分解式：p且q 为真-> p或q为真
组合式：p为真，q 为真-> p且q为真
2 选言命题：断定至少有一个为真
推理：
相容析取：p或q，p为假->q
对应语言：或者，除非 否则
真傎表：当其中一个肢命题为真时，复合命题为真
不相容析取：p或q，p为真->q为假；p或q，p为假->q为真
对应语言：要么 要么
真傎表：当一真一假时，复合命题为真
3 假言命题：断定条件关系
推理：
充分条件：（p->q， p ）->q；(p->q，非q )->非p
对应语言：如果 那么，只要 就
真值表：前真后假时为假
必要条件：（p<-q，q）->p；（p<-q，非p）->非q
对应语言：只有 才，不 就不，
真值表：前假后真时为假
4 负命题：对原命题的否定
对应语言：并非
推理：
性质命题负命题：逻辑方阵矛盾关系
联言命题负命题：非（p且q）<->非p或非q
选言命题负命题：非（p或q）<->非p且非q
假言命题负命题：充分 非（p->q）<->p且非q；必要 非（p<-q）<->非p且q
5 二难推理
（p->q，r->q，p或r ）-> q
（p->q，p->r，非q或非r）->非p

（p->r,q->s,p或q）->r或s
（p->q,r->s,非q或非s）->非p或非r

模态命题
1 真值模态命题：必然，可能

矛盾关系：必有一真一假
反对关系：必有一假。如果一个为真，另一个就为假
下反对关系：必有一真。如果一个为假，另一个就为真
差等关系：上真下真，下假上假。

并非必然p = 可能非p
并非必然非p = 可能p
并非可能p = 必然非p
并非可能非p = 必然p

2 规范模态命题：必须，允许


并非必须p = 允许非p
并非必须非p = 允许p
并非允许p = 必须非p
并非允许非p = 必须p

（二）归纳推理
归纳推理：实质是建立条件关系
1 完全归纳
全部个体，每一个都具有该种性质，因此得出结论：该类对象都具有某种性质。
2 不完全归纳
枚举归纳：所碰到的部分对象都具有该种性质，没有发现反例，因此得出结论：该类对象都具有某种性质。
科学归纳：不是碰到一个就考察一个，而是只对典型的个体考察，通过对这部分对象的分析，得出结论：该类对象都具有某种性质
3 因果推理
求同法：只有一个条件相同，其他条件均不同，这个相同条件就是产生相同现象的原因。
求异法：只有一个条件不同，其他条件均相同，这个不同条件就是产生不同现象的原因。
共变法：一个条件变化，其他条件均不变化，产生结果也变化，这个条件就是产生结果的原因。

（三）论证
加强论证：补充论据，建立条件关系（搭桥）
削弱论证：反驳论点，反驳论据，切断条件关系（拆桥）
